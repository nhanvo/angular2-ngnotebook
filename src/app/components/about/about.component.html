CRUD Operations

This project makes use of web Localstorage in order to save note data when any CREATE, READ, UPDATE or DELETE operation is performed. It also makes use of Javascript Moment library to format the time. For more detail on how to use Moment, please go to:

MomentJS

This project covers the following aspects of Angular 2:

  • Page routing
  • Localstorage
  • Two-way data binding with search form
  • Importing other Javscript Libraries
